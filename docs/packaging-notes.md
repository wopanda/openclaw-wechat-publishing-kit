> 个人信息文件 + 参考内容 + 可选主题 → 成稿 Markdown → 微信公众号草稿箱

## 推荐默认主入口
- `skills/wechat-publish-from-materials/`

## 可选能力
- `skills/material-to-graphic-report/`：需要飞书图文预览时再启用

## 正式发布链
- `skills/wechat-draft-publisher/`

# Input Contract

## 最小输入
- `persona_file`
- `references`

## 推荐补充
- `topic`
- `publish_intent`
- `target_audience`
- `need_feishu_preview`
- `style_override`

## persona_file 最少应包含
1. 身份定位
2. 真实经历
3. 失败教训
4. 表达偏好
5. 行动召唤 / CTA（如有）

## references 可接受形态
- 文章正文
- Markdown 笔记
- 多条材料摘录
- 飞书文档导出内容

## 注意
- 参考内容默认只借结构与论证线索
- 不默认接受“直接仿写某作者风格”

# 用户可定制模板

此目录包含所有用户可自定义的内容模板。部署时，替换相应文件即可定制系统行为，**无需修改任何代码**。

## 文件说明

| 文件 | 作用 | 是否必须修改 |
|------|------|------------|
| `persona.md` | 创作者人设（笔名、经历、CTA） | **必须** — 替换为自己的人设 |
| `writing-guide.md` | 写作框架和语言风格 | 推荐修改 |
| `title-formulas.md` | 25 种爆款标题公式 | 可选 |
| `abstract-transfer.md` | 参考文章时的迁移方法论 | 可选 |

## 如何定制

### 最简定制（仅替换人设）

编辑 `persona.md`，填入自己的：
- 笔名
- 内容定位（主题方向）
- 目标读者
- 个人经历锚点
- CTA（行动召唤文案）

### 完整定制

1. 修改 `persona.md` — 替换人设
2. 修改 `writing-guide.md` — 调整写作风格、文章结构
3. 修改 `title-formulas.md` — 增减或修改标题公式

## 注意事项

- 修改后无需重启服务（每次生成时实时读取）
- 保持 Markdown 格式，标题层级不要随意删改
- `persona.md` 中的 `## 基础信息`、`## 个人经历锚点`、`## 推广品牌 / CTA` 三个章节是必须保留的结构
